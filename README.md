# Ollie

React Native/Expo app connecting teens with neighbors for tasks.

## Setup
1. npm install
2. Copy .env.example to .env.local
3. Add Supabase credentials
4. npx expo start

## Tech Stack
- React Native + Expo (TypeScript)
- Supabase (Backend)
- NativeWind (Styling)
- React Query + Zustand (State)
